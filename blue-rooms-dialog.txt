      "rooms": ["101", "103", "112"]
    },
    {

      "rooms": ["201", "203"]

   - shouldBeBlue() - проверяет, должно ли помещение быть синим
   
3. Модифицирована функция updateRoomStatus() для поддержки синих помещений

4. Обновлены CSS стили для синих помещений в styles.css

5. Обновлена легенда в index.html для отображения информации о синих помещениях

6. Обновлена документация в README.md с описанием нового функционала

Теперь вы можете настраивать, какие помещения должны отображаться синим цветом, просто редактируя файл blue-rooms-config.json.
